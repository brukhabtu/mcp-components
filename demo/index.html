MCP UI Components

A composable Lit web components library for building MCP interfaces

TypeScript Accessible Dark Mode Tree-shakeable

Installation

Install the package via npm and import the components you need.

npm install @mcp/ui-components
import '@mcp/ui-components';

Buttons

Versatile button component with multiple variants, sizes, and states.

Primary Secondary Outline Ghost Destructive
Small Medium Large
Loading Disabled
<mcp-button variant="primary">Click me</mcp-button> <mcp-button variant="outline" size="lg">Large Outline</mcp-button> <mcp-button loading>Saving...</mcp-button>

Inputs

Form input components with labels, validation, and error states.

<mcp-input label="Email" type="email" placeholder="Enter email"></mcp-input> <mcp-input label="Error" error="Invalid input"></mcp-input> <mcp-textarea label="Notes" rows="4"></mcp-textarea>

Badges

Small status indicators and labels for categorization.

Default Primary Success Warning Error
Outline With Dot
<mcp-badge variant="success">Active</mcp-badge> <mcp-badge variant="warning" outline>Pending</mcp-badge> <mcp-badge variant="error" dot>Offline</mcp-badge>

Cards

Container component for grouping related content.

Basic Card

A simple card with header.

 Elevated

Card with shadow effect.

 Interactive

Hover and click me!
<mcp-card elevated> <span slot="header">Title</span> <p>Card content here</p> <mcp-button slot="footer">Action</mcp-button> </mcp-card>

Stack

Flexible layout for horizontal and vertical arrangements.

HStack Item 1 Item 2 Item 3
<mcp-hstack gap="var(--mcp-space-3)" align="center"> <mcp-button>Left</mcp-button> <mcp-button>Right</mcp-button> </mcp-hstack> <mcp-vstack gap="var(--mcp-space-2)"> <mcp-input label="Top"></mcp-input> <mcp-input label="Bottom"></mcp-input> </mcp-vstack>

Alerts

Contextual feedback messages for user actions.

This is an informational message. Your changes have been saved. Please review before continuing. Something went wrong. Click to dismiss.
<mcp-alert variant="success" title="Saved"> Changes saved successfully! </mcp-alert> <mcp-alert variant="error" dismissible> An error occurred </mcp-alert>

Loading

Spinners and progress indicators for async operations.

<mcp-spinner size="lg"></mcp-spinner> <mcp-progress value="75" show-value></mcp-progress> <mcp-progress indeterminate></mcp-progress>

Code Block

Display formatted code snippets with copy functionality.

{ "tool": "get_weather", "arguments": { "location": "San Francisco" }, "status": "success", "result": { "temperature": 68, "condition": "sunny" } }
<mcp-code language="json"> {"key": "value"} </mcp-code>

Key-Value

Display labeled data pairs in a structured format.

Server Status
<mcp-kv> <mcp-kv-item key="Status" value="Active"></mcp-kv-item> <mcp-kv-item key="ID" value="abc-123" mono copyable></mcp-kv-item> </mcp-kv>

List

Organized lists with interactive items and rich content.

Active Ready Offline
<mcp-list> <mcp-list-item title="Item" description="Details" interactive> <mcp-badge slot="suffix">Tag</mcp-badge> </mcp-list-item> </mcp-list>

Empty State
